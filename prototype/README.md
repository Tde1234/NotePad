# NotePad应用高保真原型

本目录包含NotePad应用的高保真可交互原型，基于PRD文档中的功能需求实现。

## 原型文件结构

- `notes_list.html` - 笔记列表界面
- `note_editor.html` - 笔记编辑界面
- `category_manager.html` - 分类管理界面

## 功能实现

### 笔记列表界面
- 搜索功能：顶部搜索框支持实时搜索
- 分类筛选：侧边栏分类列表可点击筛选
- 笔记展示：支持普通笔记和待办笔记显示
- 时间戳显示：每条笔记显示创建/修改时间
- 交互操作：点击笔记进入编辑，长按显示上下文菜单

### 笔记编辑界面
- 标题和内容编辑
- 分类选择和管理
- 代办功能设置（包括完成状态和提醒时间）
- 保存功能
- 更多选项（字体大小、主题设置、分享、导出、删除）

### 分类管理界面
- 分类列表展示
- 添加新分类（设置名称和颜色）
- 编辑现有分类
- 删除分类（支持左滑删除手势）
- 分类拖动排序（演示功能）

## 页面间导航逻辑

- 从笔记列表进入编辑界面：点击任何笔记项或新建笔记按钮
- 从笔记列表进入分类管理：点击右上角分类管理图标
- 从编辑界面返回列表：点击左上角返回按钮
- 从编辑界面进入分类管理：点击分类选择按钮后再点击"管理分类"
- 从分类管理返回列表：点击左上角返回按钮

## 如何测试

1. 在浏览器中打开 `notes_list.html` 开始体验
2. 所有交互元素（按钮、输入框、列表项等）都已实现点击或触摸事件
3. 浮层/模态框已实现平滑的动画效果
4. 所有页面间导航都通过相对路径实现

## 技术说明

- 使用HTML5和CSS3构建界面
- 使用Tailwind CSS v3实现响应式布局和样式
- 使用Font Awesome图标增强视觉效果
- 使用原生JavaScript实现交互逻辑和页面导航
- 适配移动设备屏幕尺寸和触摸操作

## 注意事项

- 原型中的数据均为模拟数据
- 所有操作的反馈（如保存成功提示）使用alert模拟
- 部分功能（如拖拽排序）为简化演示版本
- 建议在移动设备视图或浏览器开发者工具的移动设备模拟模式下测试
